# RareDX AI

Rare disease diagnosis and gene prioritization using HPO terms, knowledge graphs, and GCN models.

## Project structure

```
project/
├── data/           # Raw, processed, and external datasets
├── src/            # NLP, graph, models, training, eval, utils
├── api/            # FastAPI service
├── demo/           # Streamlit demo
├── notebooks/      # Exploratory and experiment notebooks
└── reports/        # Figures and technical report
```

## Setup

```bash
pip install -r requirements.txt
```

## Usage

See notebooks in `notebooks/` for data exploration and model experiments.
